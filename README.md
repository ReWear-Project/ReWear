📌 PROJECT TITLE
------------------------------
ReWear – Smart Fashion Resale & Rental Platform


📖 BRIEF DESCRIPTION
------------------------------
ReWear is a modern fashion marketplace platform designed for Gen Z users where people can buy, sell, rent, and resell second-hand fashion products. The platform promotes sustainable fashion by extending the lifecycle of clothing items and reducing fashion waste.

Users can:
- Explore fashion products
- Rent outfits for events
- Sell unused clothes
- Add products to wishlist/cart
- Place and manage orders
- Experience a modern responsive UI

The project focuses on combining sustainability with technology using a modern MERN Stack architecture and Gen-Z inspired UI/UX design.

⚙ INSTALLATION & EXECUTION STEPS
----------------------------------------------

1️⃣ Clone the Repository

git clone <repository_link>


2️⃣ Open Project Folder

cd ReWear


3️⃣ Install Frontend Dependencies

npm install


4️⃣ Install Backend Dependencies

cd backend
npm install


5️⃣ Configure Environment Variables

Create a .env file inside backend folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret


6️⃣ Run Backend Server

npm run server


7️⃣ Run Frontend

Open new terminal:

npm run dev


8️⃣ Open Browser

Visit:

http://localhost:5173
